# MASTER AI LEARNING PROMPT

You are an Expert Solution Architect, Technical Trainer, Engineering Manager, and Interview Coach.

Teach me the topic: [TOPIC NAME]

Create a complete learning repository using the following structure.

# LEVEL 1 – Definition

Provide a single sentence definition.

# LEVEL 2 – Developer Explanation

Explain in 3 simple sentences:

1. What it is
2. Why it is used
3. How it works

# LEVEL 3 – Deep Learning

Answer these questions:

1. What is it?
2. Why do we need it?
3. What problem does it solve?
4. How does it work?
5. What happens if we don't have it?
6. What business value does it provide?

# LEVEL 4 – Feynman Explanation

Explain the concept:

* To a 10-year-old child
* To a beginner developer
* To a mid-level developer
* To a senior engineer
* To a solution architect
* To a business stakeholder

# LEVEL 5 – Analogy

Provide:

* Real-world analogy
* Office analogy
* E-commerce analogy

# LEVEL 6 – Whiteboard View

Create a simple ASCII diagram.

# LEVEL 7 – Architecture View

Explain:

* Where it fits in enterprise architecture
* Dependencies
* Upstream systems
* Downstream systems

# LEVEL 8 – Trade-offs

Explain:

* Advantages
* Disadvantages
* Alternatives
* When to use
* When not to use

# LEVEL 9 – Interview Preparation

Provide:

* Beginner interview answer
* Mid-level interview answer
* Senior engineer interview answer
* Solution architect interview answer

# LEVEL 10 – Business View

Explain:

* Business value
* Cost impact
* Risk impact
* Scalability impact
* Operational impact

# LEVEL 11 – Active Recall

Generate:

* 10 beginner questions
* 10 intermediate questions
* 10 advanced questions

Do not provide answers immediately.

# LEVEL 12 – Active Recall Answers

Provide answers separately.

# LEVEL 13 – Visual Memory Aids

Create:

* Mind map
* Memory hooks
* Mnemonics

# LEVEL 14 – Portfolio Usage

Explain:

* How to use this concept in a FastAPI + Angular + PostgreSQL E-Commerce project
* Where it appears in architecture documents
* Which interview questions it supports
* Which recruiter keywords it satisfies

# LEVEL 15 – Solution Architect Notes

Provide:

* Common mistakes
* Best practices
* Enterprise recommendations
* Production considerations
* Scalability considerations

Format everything in Markdown and make it suitable for printing as a personal Technical Solution Architect Learning Handbook.
